ProtOS Distribution Package Management

Update Distribution for the ProtOS protogen visor

Updates are privately signed. This visor verifies every image offline against a key burned in at manufacture. No server is consulted and no account is checked, this prevents bad actors from sniffing out Signing keys so that our customers stay private

API

  • /api/v1/manifest.json Current release
  • /api/v1/manifest.json.minisig Signature
  • /api/v1/protos-release.pub Public key, audit only
